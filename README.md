# Simple wishlist app 
- Made with Kotlin and Jetpack compose.
- Permanently store the Wish entries in a Room database and implement CRUD functionalities.

## Tools:
- Kotlin, Jetpack Compose
- Room database
- Data Access Objects (DAO)
- Repository
- ViewModel
- Dependancy Injection
